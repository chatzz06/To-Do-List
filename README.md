
# 📝 To-Do List Web App

A simple, responsive To-Do List application built using HTML, CSS, and JavaScript.  
This project helps users manage daily tasks efficiently.

---

## 🚀 Features
- ➕ Add new tasks
- ✔ Mark tasks as completed
- 🗑 Delete tasks
- ⚡ Fast and lightweight UI
- 📱 Responsive design (works on mobile & desktop)

---

## 🛠️ Built With
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)

---

project/

1. Download or clone this repository  
2. Open the folder in **:contentReference[oaicite:0]{index=0}**
